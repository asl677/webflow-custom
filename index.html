Custom Scripts for Alex Lakas Portfolio

This repository contains custom scripts and styles for the Alex Lakas portfolio website.

Available Files:

  • animations.js - Custom animations and interactions

How to Use:

Add the following script to your Webflow site's custom code:

https://asl677.github.io/webflow-custom/scripts/animations.js

Or use the jsDelivr CDN URL:

https://cdn.jsdelivr.net/gh/asl677/webflow-custom@main/scripts/animations.js
